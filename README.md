# Brain-tumor-detection
This project focuses on automatic brain tumor classification from MRI images using deep learning techniques. Pre-trained convolutional neural networks such as VGG16, ResNet50, and MobileNetV2 are utilized through transfer learning to achieve efficient and accurate predictions.

The workflow includes data loading, preprocessing, augmentation, model training with fine-tuning, and deployment using Flask. Users can upload MRI images via a web interface and receive predicted tumor type along with confidence scores.

Features
Multi-model support (VGG16, ResNet50, MobileNetV2)
Image preprocessing and augmentation
Fine-tuning of pre-trained models
Real-time prediction with confidence score
Web deployment using Flask

Workflow
Load MRI image dataset from directory structure
Apply preprocessing and augmentation
Train model using transfer learning
Evaluate predictions
Deploy model via web interface
